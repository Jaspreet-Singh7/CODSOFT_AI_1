## 📝 Task Description
Build a simple chatbot that responds to user inputs based on predefined rules. Use if-else statements or pattern matching techniques to identify user queries and provide appropriate responses. This gives a basic understanding of natural language processing and conversation flow.

---

## ✅ What I Built
A fully functional rule-based AI chatbot with **15+ conversation intents**, **regex pattern matching**, **real-time clock/date responses** and a clean terminal interface.

### 🧠 How It Works
```
User Input → Lowercase Conversion → Regex Pattern Matching → Intent Detection → Random Response Selection → Output
```

### 💬 Supported Intents
| Intent | Example Input | Example Response |
|--------|--------------|-----------------|
| Greetings | "Hello!" | "Hello! 👋 I'm AI-Bot!" |
| How are you | "How are you?" | "I'm doing great!" |
| Name | "What's your name?" | "I'm AI-Bot!" |
| Creator | "Who made you?" | "Jaspreet Singh!" |
| Time | "What time is it?" | "It's 07:30 PM ⏰" |
| Date | "What's today's date?" | "Sunday, 14 June 2026 📅" |
| Jokes | "Tell me a joke!" | "Why do programmers..." |
| AI Topics | "Tell me about AI" | "AI is fascinating!" |
| Python | "I love Python!" | "Python is amazing!" |
| Fun Facts | "Tell me a fact" | "Did you know..." |
| Help | "What can you do?" | Lists all features |
| Thanks | "Thank you!" | "You're welcome!" |
| Goodbye | "Bye!" | "Goodbye! 👋" |
| Weather | "What's the weather?" | Redirects to web |
| Default | Unknown input | Asks to rephrase |
